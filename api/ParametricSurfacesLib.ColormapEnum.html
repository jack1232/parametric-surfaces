Enum ColormapEnum

Namespace
ParametricSurfacesLib
Assembly
ParametricSurfacesLib.dll

ColormapEnum enumeration. It defines over 60 commonly used colormaps, including commonly used names: jet, hsv, hot, cool, spring, summer, autumn, winter, bone, cooper, greys, etc. If combining these predefined colormaps with the SurfaceData.ColormapReverse field, you can get over 120 colormaps.
